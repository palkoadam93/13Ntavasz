Az internet szolgáltatásai

Ismerd meg az alapvető online szolgáltatásokat

E-mail Web DNS FTP VoIP Felhő

E-mail

Az e-mail (elektronikus levél) lehetővé teszi üzenetek küldését és fogadását az interneten keresztül. Gyors, olcsó és világszerte használható kommunikációs forma.

Web (WWW)

A World Wide Web (WWW) az internet egyik legismertebb szolgáltatása, amely weboldalak és webalkalmazások elérését biztosítja böngészők segítségével.

DNS

A Domain Name System (DNS) az internet "telefonkönyve", amely a domain neveket (pl. google.com) IP-címekké alakítja.

FTP

A File Transfer Protocol (FTP) fájlok átvitelére szolgál számítógépek között az interneten keresztül.

VoIP

A Voice over IP (VoIP) technológia lehetővé teszi hanghívások lebonyolítását interneten keresztül, például online telefonálás formájában.

Felhőszolgáltatások

A felhőszolgáltatások lehetővé teszik adatok tárolását és alkalmazások futtatását távoli szervereken, így bárhonnan elérhetők.
